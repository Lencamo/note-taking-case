## vue3-express-webpage

&emsp;&emsp;基于 vue3 官方文档 [服务端渲染（SSR）]说明搭建的一个实战示例。

&emsp;&emsp;本示例项目用到了

- webpack
- express
- Vue3
- Vue-Router@4
- Pinia

&emsp;&emsp;项目使用

```sh
# 先
npm run build:server
npm run build:client

# 然后
npm run start
```
